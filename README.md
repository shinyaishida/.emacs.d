emacs.el
====

## Install

```bash
% git clone https@github.com:shinyaishida/emacs.git ~/.emacs.d
% ~/.emacs.d/setup.sh
```
or
```bash
% mkdir <somewhere/you/wanna/put/it>
% cd !$
% git clone https@github.com:shinyaishida/emacs.git
% emacs/setup.sh

## Usage

```bash
% .emacs.d/inits/inits-manager help
```
